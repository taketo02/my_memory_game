# 神経衰弱 (Memory Game)

データサイエンス実践演習 A・第6回 ── `git clone` 体験用のサンプルリポジトリです。

## 遊び方

1. このリポジトリを `git clone` で 手元に 持ってくる
2. フォルダの 中の `index.html` を ブラウザで 開く
3. カードを 2 枚 クリックして、 同じ 絵柄を 見つける
4. 全 8 ペアが 揃ったら クリア

手数と 時間が 表示されます。 ベスト記録を 目指して みてください。

## ファイル構成

```
memory_game/
├── index.html   ── 画面の レイアウト
├── style.css    ── 見た目の デザイン
├── script.js    ── ゲームの ロジック
└── README.md    ── この ファイル
```

## いじって 遊んでみる

### 絵柄を 変える

`script.js` の 先頭にある `SYMBOLS` 配列を 書き換えると、 カードの 絵柄が 変わります。

```javascript
const SYMBOLS = ['🍣', '🍕', '🍜', '🍔', '🍩', '🍪', '🍎', '🍇'];
```

たとえば 動物に 変えてみる:

```javascript
const SYMBOLS = ['🐶', '🐱', '🐰', '🦊', '🐼', '🐨', '🐯', '🦁'];
```

8 個の 絵柄を 用意すれば、 何でも OK です。

### 色を 変える

`style.css` の 上の方に カラー指定が あります。

- `.card-back` ── カードの 裏面 (?マークの 面)
- `.card-front` ── カードの 表面 (絵柄の 面)
- `.card.matched .card-front` ── 揃った カードの 色 (緑)

### 編集した あとは

```bash
$ git status            # 何が 変わったか 確認
$ git diff              # 具体的な 変更内容
$ git add .             # 変更を ステージに 上げる
$ git commit -m "..."   # コミット
```

自分の GitHub アカウントに push すれば、 自分仕様の 改造版を 公開できます。
